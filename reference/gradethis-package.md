# gradethis: Automated Feedback for Student Exercises in 'learnr' Tutorials

Pairing with the 'learnr' R package, 'gradethis' provides multiple
methods to grade 'learnr' exercises. To learn more about 'learnr'
tutorials, please visit <https://rstudio.github.io/learnr/>.
